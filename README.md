# 🚀 Smart Traffic AI

AI-powered traffic prediction and intelligent routing system for smart cities.

## 🔥 Features
- Traffic prediction using ML
- Smart routing (AI + real-time traffic)
- Interactive map (Leaflet)
- ETA calculation
- Backend API (FastAPI)

## 🧠 Tech Stack
- Python (FastAPI, scikit-learn)
- React (Leaflet)
- OpenStreetMap (OSRM)
- Machine Learning

## 📊 How it works
1. User selects start and destination
2. Backend calculates route
3. AI predicts traffic conditions
4. System returns optimized route + ETA
